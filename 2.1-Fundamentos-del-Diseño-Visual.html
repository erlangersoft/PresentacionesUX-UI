2.1.1. Principios de Gestalt

Premisa Fundamental

"El todo es mayor que la suma de sus partes"

El cerebro humano tiende a percibir patrones completos en lugar de elementos individuales aislados.

Origen: Psicólogos alemanes del siglo XX
Max Wertheimer, Kurt Koffka, Wolfgang Köhler

Ley de Proximidad

Principio: Los elementos cercanos entre sí son percibidos como relacionados o pertenecientes al mismo grupo funcional.

Aplicación en UI:

  • Agrupación de campos de formulario
  • Espaciado entre secciones
  • Relación etiquetas-inputs
  • Organización de tarjetas

Ley de Similitud

Principio: Los elementos que comparten características visuales similares son percibidos como relacionados.

  • Consistencia en botones de acción primaria (mismo color)
  • Iconografía uniforme para acciones similares
  • Estilos tipográficos coherentes para niveles jerárquicos
  • Tratamiento visual consistente para estados de elementos

2.1.2. Jerarquía Visual y Peso Gráfico

"La jerarquía visual es la organización y presentación de elementos en orden de importancia, influyendo en la secuencia en que el usuario procesa la información."

— Edward Tufte

Peso gráfico: Determina cuánta atención visual atrae cada elemento basándose en sus propiedades visuales.

Factores que Determinan el Peso Visual

1. TAMAÑO

Los elementos más grandes atraen más atención visual.

2. COLOR Y CONTRASTE

Colores brillantes y alto contraste = Mayor peso visual.

3. POSICIÓN

Superior izquierda recibe mayor atención inicial (patrón F).

4. ESPACIADO

Elementos rodeados de espacio ganan prominencia.

5. ESTILO TIPOGRÁFICO

Bold > Semibold > Regular > Light.

6. COMPLEJIDAD

Mayor detalle atrae más atención (usar con moderación).

Escala Jerárquica Tipográfica

Título Principal (H1): 48-60px
Subtítulo (H2): 36-42px
Subtítulo Secundario (H3): 24-30px
Cuerpo de Texto: 16-18px
Texto Secundario: 14px

Ratio recomendado:

  • Entre H1 y H2: 1.414 (√2)
  • Entre H2 y H3: 1.333 (razón áurea aproximada)
  • Entre H3 y cuerpo: 1.250

✓ Dinamismo

Contraste

Definición: Diferencia visual entre elementos para crear jerarquía, legibilidad y puntos focales.

✓ Contraste Alto

Texto legible sobre fondo oscuro

Ratio: 12.6:1 (AAA)

✗ Contraste Bajo

Texto difícil de leer

Ratio: 2.1:1 (Falla)

Estándares WCAG 2.1:
  • Nivel AA: 4.5:1 texto normal, 3:1 texto grande
  • Nivel AAA: 7:1 texto normal, 4.5:1 texto grande

Técnicas de Énfasis

1. AISLAMIENTO

Rodear elementos de espacio en blanco.

2. COLOR DE ACENTO

Regla 60-30-10 para distribución cromática.

3. PROFUNDIDAD

Uso de sombras para crear elevación.

4. MOVIMIENTO

Animaciones estratégicas (usar con moderación).

⚠️ Advertencia: El exceso de énfasis equivale a la ausencia de énfasis.

Ritmo Visual

Definición: Repetición y variación de elementos visuales que crea sensación de movimiento y flujo.

RITMO REGULAR

Predecible y ordenado

RITMO PROGRESIVO

Guía el flujo visual

Técnica 80/20:

80% consistencia y repetición + 20% variación e innovación

Caso Práctico: EduConnect

Problema Identificado

Sobrecarga visual en dashboard principal que causa:

  • Ausencia de jerarquía clara
  • Elementos compitiendo por atención
  • Espaciado inconsistente
  • Falta de agrupación visual

Resultado: 66% de usuarios abandonan en el primer mes

Solución Aplicando Fundamentos Visuales

Aplicación de Gestalt

  • Proximidad: Reagrupación en zonas funcionales
  • Región común: Cards para contenido relacionado
  • Similitud: Consistencia en botones CTA

Jerarquía Estratégica

  • Nivel 1: Curso actual (H2 32px, verde brand)
  • Nivel 2: Recomendaciones (H3 24px)
  • Nivel 3: Actividad reciente (16px)
Métricas Esperadas:
  • Tiempo hasta primera acción: -40% (4.2min → 2.5min)
  • Tasa de inicio primer curso: +47% (34% → 50%)
  • Satisfacción visual: +24% (3.4/5 → 4.2/5)

Conclusiones Clave

  • Los principios Gestalt son fundamentales para crear interfaces intuitivas
  • La jerarquía visual guía efectivamente la atención del usuario
  • El balance y contraste crean armonía y legibilidad
  • El espaciado es tan importante como los elementos mismos
  • La proporción áurea y sistemas modulares crean armonía natural
  • El ritmo visual mantiene engagement sin monotonía
